2 - Exercício: Seleção de Gênero Musical Preferido

Desenvolva um script em JavaScript utilizando SWITCH que permita ao usuário escolher seu gênero musical preferido a partir de uma lista pré-definida. O programa deve exibir uma lista numerada de opções de gêneros musicais e solicitar que o usuário insira o número correspondente ao seu gênero favorito.

As opções de gêneros musicais e seus respectivos números são os seguintes:

1 - Rock
2 - Pop
3 - Hip Hop
4 - Eletrônica
5 - Jazz
6 - Clássica

O programa deve tratar casos em que o usuário insira uma opção inválida, exibindo uma mensagem apropriada.

Após a seleção do usuário, o programa deve exibir a seguinte mensagem na página: "Seu gênero musical preferido é: [gênero selecionado]".

3 - Exercício: Criação de Script para Seleção de seu Tema escolhido.

Desenvolva um script em JavaScript utilizando a estrutura SWITCH que permita criar e selecionar seu próprio tema de interesse para desenvolver um programa relacionado. O programa deve orientar o usuário a inserir um número correspondente ao seu tema escolhido, e após a seleção, exibir uma mensagem personalizada relacionada ao tema.

Após a seleção do tema, o programa deve exibir uma mensagem correspondente ao tema escolhido.

O programa deve tratar casos em que o usuário insira uma opção inválida, exibindo uma mensagem apropriada.

Após a seleção do usuário, o programa deve incentivar o aluno a explorar e desenvolver habilidades na área do tema escolhido.
